# Zeppelin 0.7.2 中文文档

+   [在线阅读](https://zeppelin.flygon.net)
## 下载

### Docker

```
docker pull apachecn0/zeppelin-doc-zh
docker run -tid -p <port>:80 apachecn0/zeppelin-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g zeppelin-doc-zh
zeppelin-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
